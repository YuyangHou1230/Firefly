## 安装ollma
```bash
irm https://ollama.com/install.ps1 | iex
```

## 部署deepseek-r1 7b
```bash
ollama pull deepseek-r1:7b
```
测试模型
```bash
ollama run deepseek-r1:7b
```

部署千问
```bash
ollama pull qwen3.5:9b
```
